Para instalar las proptypes en yarn
    yarn add prop-types
En npm
    npm install prop-types

                    Jest

Para instalar la libreria Jest para test
    yarn add --dev jest
En npm
    npm- install --save-dev jest
Para continuar con la configuracion al package.json hay que agregar en el script
    "scripts": {
        "test": "jest --watchAll"
    }
Luego para iniciar un "test" seria con
    yarn test
Para instalar los snippets de Jest por terminal
    yarn add -D @types/jest
Ante el error de ECMAScript hay que terminar de configurar para yarn
    yarn add --dev babel-jest @babel/core @babel/preset-env
Luego crear archhivo 'babel.config.js' si no viene instalado, agregar:
    module.exports = {
        presets: [['@babel/preset-env', {targets: {node: 'current'}}]],
    };
Si aparece el error "You appear to be using a native ECMAScript module configuration file, which is only supported when running Babel asynchronously."
    Cambiar el nombre de la extension a .cjs
Cuando quiera hacer test en archivos que poseen promesas async await es necesario crear ambos archivos: 'jest.config.js' y 'jest.setup.js', con las configuraciones dentro
Luego es necesario ejecutar en la terminal lo siguiente:
    yarn add -D whatwg-fetch

                    React Testin Library
Para instalar hay que jecutar el sig. comando
    yarn add --dev @testing-library/react
    yarn add -D jest-environment-jsdom
    yarn add -D @babel/preset-react
